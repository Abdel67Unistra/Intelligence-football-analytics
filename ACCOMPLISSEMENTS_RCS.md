## 📈 MÉTRIQUES CLÉS RCS

### **🎯 Expected Goals (xG)**
```python
# Formule adaptée style RCS
xG = xG_base × bonus_action × bonus_joueur

# Bonus spéciaux
contre_attaque: +35%
Emanuel Emegha: +15%
Dilane Bakwa:   +10%
Habib Diarra:   +8%
```

### **📊 PPDA (Pressing)**
```python
PPDA = passes_adverses / actions_defensives
Objectif RCS: 8-12 (pressing moyen-haut coordonné)
```

### **💰 Budget Scouting**
